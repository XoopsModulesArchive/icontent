Manuel d'utilisation - Administration des dossiers

 

Voici une copie d'écran typique de la section administration des dossiers :

 

I Premier tableau - "Dossiers choisis"

Tout en haut de l'image vous pouvez apercevoir un tableau intitulé "Dossiers choisis" il contient la liste des dossiers depuis la racine "inPages" que vous avez successivement choisis pour afficher le dossier courant (dont vous pouvez apercevoir le nom dans le titre du tableau suivant). Au passage le lien entre les deux tableaux est utilisé pour naviguer facilement de l'administration des dossiers à l'administration des pages dans le même dossier.

 

II Deuxième tableau - "Dossier courant"

Dans le deuxième tableau est inscrit le nom du dossier courant (ligne bleue foncée). En dessous à gauche est affichée la liste des dossiers contenus dans le dossier courant et, j'insiste sur ce point, au préalable activés par vous. Nous reviendrons sur ce point dans la suite de la documentation. Les dossiers non activés ne seront pas affichés dans ce tableau.

A droite de ce tableau ce situent les informations sur chaque dossier se trouvant à gauche (cf. repère 1 et 2) :

  • Repère 1 : indique le nombre de dossiers et de pages activés et contenus dans le dossier. Ces éléments doivent tous les deux êtres égaux à 0 avant de pouvoir désactiver le dossier spécifié. Nous verrons comment désactiver un dossier par la suite.
  • Repère 2 : indique le nom réel du dossier à savoir le nom qu'il porte sur le serveur. En effet le nom du dossier peut différer de son nom réel puisque vous pouvez le renommer. Nous verrons comment par la suite.

Note : il est possible d'afficher l'url complète d'un dossier en laissant quelques instants la souris sur le nom du dossier. Son url physique sur le serveur apparaîtra alors.

Pour accéder au contenu d'un dossier vous n'avez qu'a cliquer sur son nom.

 

III Troisième tableau - "Actions"

Ce tableau répertorie toutes les actions que vous pouvez effectuer sur les dossiers contenus par le dossier courant.

Voici la liste des commandes :

  • Repère 3 : indique la liste des dossiers non activés. Vous n'avez qu'a sélectionner les dossiers que vous voulez puis cliquer sur "Activer les dossiers" pour les activer. Il est à noter que vous pouvez en activer plusieurs à la fois soit en les sélectionnant tous, soit en choisissant seulement quelques uns grâce à la touche SHIFT de votre clavier.
  • Repère 4 : indique les dossiers que vous pouvez configurer. Vous pouvez aussi en sélectionner plusieurs, comme expliqué pour le repère 3. Dans la section de configuration des dossiers (comme celle des pages d'ailleurs) vous pouvez changer le nom d'un dossier, changer les groupes qui auront le droit d'accéder à tout son contenu (y compris tous les sous dossiers et les pages contenues dans ces dossiers), enfin choisir de cacher aux visiteurs le dossier (il sera toujours affichable dans l'explorateur via son url mais n'apparaîtra pas dans le contenu de son dossier parent). Une copie d'écran détaillée est disponible dans la section configuration d'un dossier.
  • Repère 5 : indique les dossiers que vous pouvez désactiver. Remarque : seul les dossiers activés peuvent être désactivés et donc apparaissent dans cette liste.

 

IV Activer un dossier

Pour activer un dossier vous devrez au préalable activer tous ses dossiers parents si il en possède. Ensuite rendez vous grâce à l'explorateur iContent dans le dossier parent de celui que vous chercher activer. Ensuite rendez vous dans le tableau "Actions" tout à sa gauche (cf. repère 3 de l'image 1). Dans cette liste vous trouverez le dossier que vous cherchez à activer (il aura le nom que vous avez donné au dossier physique sur votre serveur). Sélectionnez le puis cliquez sur "Activer les dossiers". Et voilà votre dossier est désormais activé. Vous pouvez le configurer à loisir et changer son nom. Le nom par défaut est le nom physique du dossier sur le serveur.

 

V Configurer un dossier

Pour configurer un dossier vous devez au préalable l'activer puis naviguer jusqu'à son dossier parent, ensuite rendez vous dans le tableau "Actions" à sa colonne centrale (cf. repère 4 de l'image 1). Sélectionnez le (ou les) dossier(s) que vous voulez configurer (le nom affiché sera celui que vous aurez donné au dossier via la configuration de ce dernier) puis cliquez sur "Configurer les dossiers". Voici une copie d'écran du résultat :

  • Repère 1 : vous pouvez changer le nom du dossier dans cette partie.
  • Repère 2 : vous pouvez voir la page d'accueil du dossier (celle qui s'affichera au lieu d'afficher le contenu du dossier quand un visiteur cliquera sur le nom du dossier) et au besoin l'effacer si vous ne désirez plus de page d'accueil pour ce dossier.
  • Repère 3 : vous pouvez changer les droits d'accès de chaque groupe vers ce dossier et tout son contenu (c'est à dire tous les dossiers et pages qu'il contient). Cochez les noms des groupes à qui vous voulez laisser l'accès libre.
  • Repère 4 : vous pouvez choisir de cacher ou non le dossier dans cette partie. Un dossier caché sera toujours accessible mais n'apparaîtra plus dans le contenu de son dossier parent (seulement en dehors de l'administration).

 

VI Désactiver un dossier

Pour désactiver un dossier vous devez l'avoir au préalable activé. Désactiver un dossier ne le supprimera pas du serveur mais ne l'affichera plus dans l'explorateur iContent. Un dossier est désactivable si il ne contient plus de sous dossiers ou de pages activés. Pour désactiver un dossier vous devrez donc désactiver tous les dossiers et toutes les pages qu'il contient. Pour la désactivation proprement dite, naviguez jusqu'au dossier parent de celui que vous voulez désactiver puis rendez vous dans le tableau "Actions" à sa colonne de droite. Vous trouverez à cet endroit le nom du dossier (le nom affiché sera celui que vous aurez donné au dossier via la configuration de ce dernier) que vous souhaitez désactiver. Sélectionnez le puis cliquez sur "Désactivez les dossiers". Le dossier est maintenant désactivé.
